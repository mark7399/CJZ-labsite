# 主要研究方向与成果

我们的核心研究是**聚焦于图计算方法，深度挖掘生物网络中的关联与传播模式**，旨在：

* **提升复杂疾病诊断准确率**
* **挖掘疾病相关的早期筛查靶点**
* **推动疾病的个性化精准治疗**

---

### 研究方向一：多组学网络分析与癌症精准治疗

* **挑战：** 多组学数据（如基因、蛋白、代谢网络）的调控关系呈现跨尺度、关联弱的特性，传统方法难以发现其分子调控的异常模式。
* **我们的方法与成果：** 我们提出了创新的**高阶图匹配方法**，成功应用于分析鼻咽癌的转移模式。研究发现，通过区分血道转移和淋巴道转移，可以指导更精准的个性化治疗方案（如局部放疗 vs. 免疫治疗）。
* **临床影响力：** 该方法指导下的精准治疗决策，使复发鼻咽癌患者的**2年无进展生存率提高了20.5%**。相关成果发表在IEEE TPAMI、JBHI等期刊。

### 研究方向二：多模态脑网络分析与神经退行性疾病诊断

* **挑战：** 多模态脑网络（如结构与功能网络）是嵌入在高维流形空间中的不规则图结构，传统的欧几里得分析方法会极大破坏其拓扑信息，导致病理传播通路难以识别。
* **我们的方法与成果：** 我们开发了**图谐波流形学习**方法，能够有效识别阿尔茨海默症（AD）中的Tau蛋白病理传播关键通路。
* **临床影响力：** 该方法识别的通路与AD临床早筛核心指标（如MMSE分数）高度相关。与基准方法相比，在AD早期筛查任务中，**准确率提升25.2%，灵敏度提升42.1%，特异性提升29.5%**。相关成果发表在IEEE TMI、JBHI、IPMI等期刊会议。

### 研究方向三：多来源异构数据融合与慢性脑疾病诊断

* **挑战：** 融合来自多来源（如医学影像、临床文本、基因组学）的异构数据是巨大挑战，因为现有图计算方法难以在不同节点类型和连接属性的网络间进行有效的信息传递。
* **我们的方法与成果：** 我们设计了**图判别融合学习框架**，能够有效融合多源异构特征，显著提升了多种慢性脑疾病的诊断精度。
* **临床影响力：** 在三个大型公开数据库（ADNI, ADHD-200, MDD-meta-project）上进行验证，与临床诊断基准方法相比，我们的方法将**诊断准确率平均提高了17%**。具体来说，对注意力缺陷多动症(ADHD)提升17.9%，轻度认知障碍(MCI)提升10.3%，抑郁症(MDD)提升22.9%。相关成果发表在IEEE TNNLS、Medical Image Analysis等期刊。
